# Digital Ad Inventory Yield Optimization
**Disney Addressable Inventory Analytics Portfolio Project**

![Yield Dashboard](assets/yield_dashboard.png)

## Overview
Strategic analysis of 550,000+ ad impressions identifying $6.3M revenue recovery 
opportunity through yield optimization, pacing correction, and audience targeting 
strategies.

**Business Context:** Digital inventory management for streaming/media platform 
(simulating Hulu/Disney+ ad operations)

## Key Insights
- **$6.3M** total revenue loss identified across 5 business verticals
- **22.3%** of inventory under-pacing (below 85% delivery target)
- **Sports Vertical:** Highest risk ($1.6M loss)
- **Parents Audience:** Best performing segment (92% yield rate)

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Generate data: `python generate_data.py`
3. Run analysis: `python yield_analysis.py`
4. View outputs in `assets/` folder
